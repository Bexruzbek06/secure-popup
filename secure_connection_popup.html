Подключение защищено

Данные браузера Chrome подтверждают, что сертификат сайта выпущен компанией WR2.

Данные сертификата: соединение с sites.google.com зашифровано с помощью современных наборов шифров.

В этом подключении используется протокол QUIC.

Соединение зашифровано и проверено с помощью AES_128_GCM. В качестве механизма обмена ключами используется X25519MLKEM768.
